# TCC de Desenvolvimento de Sistemas

## Objetivo do projeto

Nosso objetivo será criar um jogo online 3D e lança-lo na plataforma [steam](https://store.steampowered.com/?l=portuguese).

## Escopo

Planejamos esse escopo para desenvolvermos o nosso TCC

- **Tipo de Aplicativo**: jogo online 3D
- **Engine**: [Godot](https://godotengine.org)
- **Linguagem de programação**: [GDScript](https://docs.godotengine.org/en/stable/tutorials/scripting/gdscript/index.html)
- **Categorias em que o aplicatiivo se encaixa**:
  - Casual PVP
  - Online
  - Roguelike
  - Física

## Gameplay

A gameplay se consistirá nos player controlando carros com o objetivo que demolir uns aos outros.
Ela será dividida em rodadas. O vencedor da rodada terá que escolher um debuff temporario, enquanto os perdedores terão de escolher um buff permanente.

## Logs

A pasta logs funcionará como um diario. A organização dessa pasta será a seguinte:

```Path
|-- logs
|   |-- {nome do aluno}
|   |   |-- {MM:DD}.md
```
